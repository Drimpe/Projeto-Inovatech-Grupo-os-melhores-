- NO MENU À ESQUERDA (PASTA ABERTA) SELECIONE CLIQUE NO ARQUIVO INDEX.HTML

- CLIQUE NO BOTÃO "RUN LIVE SERVER" LOCALIZADO NO CANTO INFERIOR DIREITO DO VSCODE
